Odoo 16 Journal Sequence

Bring backs the journal sequence concepts from Odoo 13. Now user can easily configure and change the invoice/bill sequence number from the journals. Download the Module and Install it.

Features

  • Entry and Refund Sequence For Journals.
  • Customize sequence as per need.
  • Works as in Odoo 13.

Invoicing/Accounting -> Configuration -> Journals

Creating new journal

Once Journal is saved, the sequence and refund sequence will get automatically generated.

Customize Sequence as per the need by clicking the sequence field
